# 🚀 Netlify 部署快速指南

## 📋 **GitHub + Netlify 集成部署 (推荐)**

### 步骤1: 连接GitHub
1. 打开 https://netlify.com
2. 点击 "Sign up" 注册
3. 选择 "GitHub" 登录

### 步骤2: 选择仓库
1. 点击 "New site from Git"
2. 选择 "GitHub"
3. 选择仓库: `kaicao/oc_aptdb`
4. 选择分支: `main`

### 步骤3: 配置部署
- **Base directory**: `deployment` (如果需要)
- **Build command**: 留空 (静态文件无需构建)
- **Publish directory**: `deployment` 或 `dist`

### 步骤4: 部署
1. 点击 "Deploy site"
2. 等待30秒部署完成
3. 获得您的链接: `https://random-name-123456.netlify.app`
