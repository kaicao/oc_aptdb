# 🚀 Netlify 部署操作指南

## ✅ **部署文件已准备**
- 📁 deployment/ 文件夹：完整前端应用
- 📄 index.html：主应用文件
- 📦 assets/：CSS和JS文件
- ⚙️ netlify.toml：Netlify配置文件

## 🎯 **立即部署步骤 (30秒)**

### 步骤1: 访问Netlify
1. 打开浏览器访问：https://netlify.com
2. 点击 "Sign up" 注册账户
3. 选择 "Sign up with GitHub"

### 步骤2: 连接GitHub仓库
1. 用GitHub登录
2. 点击 "New site from Git"
3. 选择 "GitHub"
4. 选择仓库：`kaicao/oc_aptdb`
5. 选择分支：`main`

### 步骤3: 配置部署设置
- **Base directory**: `deployment` (或留空)
- **Build command**: 留空 (静态文件无需构建)
- **Publish directory**: `deployment` (或 `./`)

### 步骤4: 部署
1. 点击 "Deploy site"
2. 等待30秒自动部署
3. 获得您的链接：`https://random-name-123456.netlify.app`
